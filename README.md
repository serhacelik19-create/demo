## 💡 Developer's Journey, Adaptability & Vision

### 🚀 Rapid Technological Adaptation
One of the core philosophies behind the development of these projects is **continuous learning and rapid adaptation**. Several components, structures, and programming paradigms present in this repository were built using languages, frameworks, and tools that I had little to no prior familiarity with. 
Instead of seeing unfamiliar stacks as obstacles, I approached them as opportunities to learn, grow, and demonstrate that a software engineer's most valuable asset is the ability to acquire new skills on the fly.

### 🤖 Collaborative AI & Agentic Workflows
All projects here are built using **modern AI-assisted development methodologies (Vibe Coding)**. By collaborating closely with advanced LLMs, coding agents, and autonomous AI orchestration tools:
- I transformed conceptual designs into structured, production-ready boilerplates rapidly.
- I bridged technical gaps in syntax, environment configurations, and dependency integration.
- I maintained clean architectural patterns, demonstrating how developers can act as systems orchestrators and architects alongside AI.

This repository is a testament to what can be accomplished when an insatiable hunger for learning meets the power of collaborative AI engineering.

## 📁 Repository Structure

This repository is structured as a multi-project workspace, containing independent projects with distinct stacks:

### 1. [kortex](./kortex)
An **AI-Powered Educational Platform** designed to revolutionize how students prepare for exams, practice questions, and monitor their learning progress.
* **backend**: Node.js/TypeScript backend API.
* **panel**: React/TypeScript-based management and monitoring console.
* **mobile-app**: Flutter mobile application designed for students and parents.
* *For setup and run instructions, please refer to the [Kortex README](./kortex/README.md).*

### 2. [dicle-university-template](./dicle-university-template)
A comprehensive **University Information Portal & Clinic CMS Template** custom-designed for Dicle University. 
* **Framework**: Next.js 15 (App Router, Turbopack) & React Server Components.
* **Database**: PostgreSQL with Prisma ORM.
* **Features**: Dynamic student services portal, academic unit manager, administrative doctor staff directory, emergency polyclinics manager, automatic PDF food menu parser, and an interactive AI virtual assistant.
* *For setup and run instructions, please refer to the [Dicle University Template README](./dicle-university-template/README.md).*

### 3. [chefvision](./chefvision)
A smart recipe advisor and ingredient recognition system named **ChefVision AI**, powered by camera scans and advanced voice assistant interactions.
* **mobile-app**: Flutter client with camera scanning, local notifications, secure storage, and voice commands.
* **backend**: Python (FastAPI) back-end integrated with PostgreSQL database, Redis caching, Firebase authentication endpoints, and Gemini AI.
* **dashboard**: Next.js (App Router) web dashboard for admin oversight.
* *For setup and run instructions, please refer to the [ChefVision README](./chefvision/README.md).*
